      About This Project

      This is a mini project built as part of the 4th semester curriculum. The goal was to build something genuinely useful for tomato farmers using AI — and keep it simple enough to actually finish.

      🧠 AI Model

      MobileNetV2 pre-trained on ImageNet, fine-tuned on the PlantVillage dataset (tomato subset). The model outputs a disease class and a confidence score. If confidence is below 70%, the app warns you to consult an expert instead of acting on the result.

      📦 Dataset

      PlantVillage dataset — freely available on Kaggle. The tomato subset contains approximately 18,000 leaf images across 10 classes (9 diseases + healthy). Training was done on Google Colab using a free GPU.

      🌦 Weather Feature

      Uses the OpenWeatherMap free-tier API. This is not ML weather prediction — it fetches real forecast data and applies rule-based thresholds drawn from agricultural literature to estimate disease risk.

      🪱 Soil Feature

      A simple lookup-table approach. The user enters soil readings (pH, moisture, NPK on a 1–5 scale), and the app compares them against known ideal ranges for tomato cultivation, flagging issues with specific remedies.
